# site-pgent-pages

## 项目简介

`site-pgent-pages` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，而是作为一组静态页面的存放与展示空间，方便集中管理和版本控制。

## 目录说明

```
site-pgent-pages/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的公共资源（CSS、JS、图片等）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 页面独立存放，命名清晰，便于检索。
- `assets/`：公共资源目录，避免页面间重复引用。

## 页面归档说明

- 所有 HTML 页面均为独立文件，可直接通过浏览器打开。
- 页面内容可能包含演示、工具、文档或实验性项目，不承诺长期可用或更新。
- 归档页面可能引用外部资源（如 CDN），请确保网络可访问。

## 维护说明

- 本仓库由维护者不定期更新，欢迎提交 Issue 或 Pull Request。
- 提交新页面时请放在 `pages/` 目录下，并尽量保持命名规范。
- 不建议在页面中包含敏感信息或外部依赖过重的资源。
- 如发现页面无法正常显示，可检查资源路径或提交反馈。
